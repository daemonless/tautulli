## Deployment

### Podman Compose

```yaml
services:
  tautulli:
    image: ghcr.io/daemonless/tautulli:latest
    container_name: tautulli
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
      - TAUTULLI_DOCKER=True
    volumes:
      - /path/to/containers/tautulli:/config
    ports:
      - 8181:8181
    restart: unless-stopped
```

### Podman CLI

```bash
podman run -d --name tautulli \
```
